常见问题

快速找到您需要的答案

如何开始使用 Linux Tool？

您可以通过 npm 安装 Linux Tool，然后运行初始化命令即可快速开始。查看我们的快速入门指南了解更多详情。

Linux Tool 是否免费？

是的，Linux Tool 是开源免费的工具。我们也提供企业版服务，包含更多高级功能和专业支持。

如何贡献代码？

欢迎您在 GitHub 上 Fork 项目，提交 Pull Request。请先阅读我们的贡献指南了解详细流程。

遇到技术问题怎么办？

您可以在 GitHub Issues 中搜索类似问题，或者提交新的 Issue。我们的社区和技术团队会尽快为您解答。
